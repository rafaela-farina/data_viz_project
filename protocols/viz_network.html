Country Connections Network

Visualization by Rafaela Dos Santos Farina

Overview

An interactive network visualization showing the web of direct flight connections between countries. Built with Flourish, this visualization complements the D3 map by showing the density and clustering of international aviation connections.

Visualization Tool

Platform Flourish
Chart Type Network Diagram
Public URL View on Flourish

Data Source

File country_connections.csv
Original Source OpenFlights Database
Processing Aggregated unique airport pairs per country

Data Schema

Column Type Description
source_country String Origin country name
dest_country String Destination country name
num_routes Integer Number of unique airport-to-airport routes

Design Decisions

  • • Node size represents number of connections
  • • Edge thickness indicates route density between countries
  • • Clustering algorithm groups geographically related countries
  • • Interactive hover reveals connection details
