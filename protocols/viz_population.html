Ratio of International Airports to Relative Population

Visualization by Colin Augustine

Overview

An interactive Tableau visualization comparing the number of international airports to population size across countries. This analysis reveals which regions are over-served or under-served in terms of air travel infrastructure relative to their population.

Visualization Tool

Platform Tableau Public
Chart Type Geographic Map / Scatter Plot
Public URL View on Tableau Public

Data Sources

Airport Data OpenFlights Database (airport_connectivity.csv)
Population Data World Bank / UN Population Estimates
Processing Merged airport counts with population by country

Key Metrics

Metric Description
Airport Count Number of international airports per country
Population Country population (latest available)
Ratio Airports per million inhabitants

Key Findings

  • Over-served: Small wealthy nations (Luxembourg, Iceland, Malta) have high airport-to-population ratios
  • Under-served: Large developing nations (India, Bangladesh, Nigeria) have low ratios despite large populations
  • Strategic hubs: Gulf states (UAE, Qatar) maximize connectivity through concentrated mega-hubs

Interactivity

  • • Hover over countries for detailed statistics
  • • Filter by region or population size
  • • Click to highlight specific countries
