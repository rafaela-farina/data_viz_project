Visualization Details

Interactive Elements

The section features a tab-based selector allowing users to switch between continental hubs. Each tab displays:

  • Airport name and IATA code
  • Three key statistics (countries connected, ranking, annual passengers)
  • Contextual description explaining the hub's significance

Planned Enhancement

A Flourish line/stream chart will be embedded showing monthly passenger trends for 2024, allowing comparison across all six hubs. This visualization is being developed by teammate Rafaela Dos Santos Farina.

Key Insights

Size vs. Connectivity

ATL handles the most passengers (108M) but CDG connects to more countries (115+). This highlights the difference between domestic hub dominance (ATL serves primarily US domestic routes) versus international reach (CDG leverages Air France's global network).

Strategic Positioning

Dubai (DXB) exemplifies the "super-connector" model ‚Äî positioned between Europe, Africa, and Asia, it serves as a layover hub for long-haul routes, driving its 92.3M passengers despite UAE's relatively small domestic market.

Regional Disparities

The gap between top hubs (ATL: 108M) and smaller regional leaders (CMN: 10M) reveals significant infrastructure inequality in global aviation.
